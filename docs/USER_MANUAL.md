# 使用说明书

## 一、启动与停止

### 启动
双击项目根目录下的 `start.bat`，自动完成：
1. 激活 Python 虚拟环境
2. 启动后端服务（端口 5000）
3. 启动前端开发服务器（端口 8080）

浏览器访问 **http://localhost:8080**

### 停止
双击 `stop.bat`，或关闭命令行窗口。

---

## 二、图片检测

### 操作步骤
1. 点击顶部 **"图片检测"** Tab
2. 点击 **"选择图片文件夹"** 按钮，选择包含施工图纸的文件夹
3. 系统自动加载并检测第一张图片
4. 使用 **左/右箭头** 或底部胶片条切换图片
5. 支持 **对比视图**（原图 vs 检测结果）和 **仅看结果** 两种模式

### 检测结果
右侧面板展示：
- **检测统计**: 按类别统计识别到的目标数量
- **检测列表**: 每个目标的类别、置信度
- **规范检查**: 12 条几何规则的合规/不合规/无法检测结果及总分

### 导出
- **导出图片**: 下载带标注框的检测结果图
- **导出 JSON**: 下载完整检测数据

## 三、文档分析

### 3.1 上传文档
1. 点击顶部 **"文档分析"** Tab
2. 点击 **"选择 .docx 文件"** 上传施工方案文档
3. 系统自动提取文档中的图片并展示

> 重复上传同一文件会提示复用历史分析结果，避免重复工作。

### 3.2 历史任务
上传区上方展示历史分析记录，可：
- **打开**: 恢复之前的分析任务
- **删除**: 清除任务及所有数据

### 3.3 分析图片

#### 单张分析
1. 点击左侧图片网格中的任意图片
2. 在右侧详情面板点击 **"AI 分析此图"**
3. 等待 AI 返回分析结果（类型、摘要、评估等）

#### 批量分析
1. 点击状态栏的 **"批量分析全部"** 或 **"继续批量分析"**
2. 系统自动并发分析所有待处理图片
3. 实时进度条 + 每张图状态指示

**状态指示说明**:
| 指示 | 含义 |
|------|------|
| 灰点 | 等待分析 |
| 紫色脉冲点 | 正在分析中 |
| 绿色对勾 | 分析完成 |
| 红色感叹号 | 分析异常 |

> "继续批量分析"仅重新处理失败的图片，已成功的不会重复分析。

### 3.4 查看结果

选中已分析的图片，右侧详情面板显示：
- **图纸类型**: AI 分类结果（彩色标签）
- **AI 摘要**: 图片内容详细描述
- **AI 评估**: 四维度评估（完整性/清晰度/布局合理性/规范符合性）+ 等级（优/良/中/差）
- **施工计划**: 如有进度类图纸，展示工期和里程碑
- **尺寸规格**: 提取的尺寸、型号、数量

### 3.5 汇总报告

批量分析完成后自动生成综合评估报告，包含：
- 总体概述与整体质量评级
- 图纸类型分布统计
- 质量评级分布
- 各图片评估详情
- 关键施工参数汇总（桩基规格、分区面积、设备配置等）
- AI 综合评审意见（可选）

**导出 PDF**: 点击汇总报告右上角 **"导出PDF"** 按钮，在新页面中通过浏览器"另存为 PDF"保存。

## 四、AI 分析说明

### 免责声明
> 本系统在 CV 的基础上引入大模型用于辅助审查，AI 生成可能有误，请仔细甄别。

### 分析异常处理
- 网络不稳定时部分图片可能分析失败，图片卡片会显示红色感叹号
- 状态栏会提示"待分析"数量
- 可选中异常图片点击"重新分析"，或使用"继续批量分析"
- 汇总报告的统计数据由系统自动计算，不依赖 AI，保证数字准确

### 模型配置
- 视觉模型: GLM-4.6V（主）+ GLM-4.5V（备），自动轮换
- 汇总模型: Qwen2.5-7B-Instruct（文本模型）
- 5 个 API Key 并发轮换，单 Key 限流时自动切换
